# Real-Time Chat Application

## Overview

A full-stack real-time chat application built using React.js, Node.js, Express.js, MongoDB Atlas, and Socket.io.

## Features

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Create Chat Rooms
* Join Chat Rooms
* Real-Time Messaging
* Online Users
* Typing Indicator
* Chat History
* Message Timestamps
* Join/Leave Notifications
* Logout Functionality

## Tech Stack

Frontend:

* React.js
* React Router
* Axios
* Socket.io Client

Backend:

* Node.js
* Express.js
* MongoDB Atlas
* Socket.io
* JWT
* bcryptjs

## Installation

Backend:

npm install
npm run dev

Frontend:

npm install
npm start
